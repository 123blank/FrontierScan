# Frontend Conventions

Frontend stack:

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Axios

Guidance:

- Prefer existing components and CSS patterns.
- Keep B2B admin workflows clear, table-driven, and operation-oriented.
- For UI work, run `npm run build` from `frontend`.
- Avoid introducing decorative dashboard patterns when an operational layout is sufficient.
