# FrontierScan Project Skills

This directory is reserved for project-local Codex Skills that support the FrontierScan Harness workflow.

Planned MVP Skills:

```text
frontier-common
frontier-kb-generate
frontier-kb-query
frontier-state-runner
frontier-requirement-breakdown
frontier-task-dag-planner
frontier-code-review-gate
frontier-test-gate
```

Do not add runtime workflow state here. Runtime state belongs in `.harness/`.

See `docs/harness-skill-customization-plan.md` for the Skill creation plan.
