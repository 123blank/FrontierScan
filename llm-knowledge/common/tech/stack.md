# FrontierScan Technology Stack

Backend:

- Spring Boot 3.3.5
- Java 17
- Maven
- PostgreSQL
- Redis
- Flyway
- Spring Security
- JPA and MyBatis-Plus
- Rome and Jsoup for feed/page parsing

Frontend:

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- Axios

Deployment:

- Docker Compose
- Backend and frontend Dockerfiles
