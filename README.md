# FrontierScan

FrontierScan is an enterprise-style web agent for collecting, organizing, and presenting frontier technology and AI website updates.

This first milestone provides a runnable frontend/backend skeleton:

- Backend: Spring Boot 3, Java 17, Maven
- Frontend: Vue 3, TypeScript, Vite
- Storage: PostgreSQL
- Cache/task helper: Redis
- Deployment: Docker Compose
